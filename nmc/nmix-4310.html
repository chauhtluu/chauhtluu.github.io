Midterm Project


For the midterm project in this class, I decided to create an app that someone could use for their travel plans. For this project, I incorporated some of things I learned previously and utilized new elements like a picker and form to make the app more interactive.

Final Project


Through the second half of the semester, I learned about new elements that could be used in an app that would help improve the interactivity and usability of the app. As I felt that my midterm project was lacking, I chose to expanded upon it for my final project. For this project, I primarily focused on fixing the layout of the content and the files themselves (so that the code look more organized and easier to navigate), but I also added newer features like the ability to rate a location and check off if you've been to a location (using animated buttons). I also create a new bottom sheet where when you pull it up on the screen, it will display more details about the location.
